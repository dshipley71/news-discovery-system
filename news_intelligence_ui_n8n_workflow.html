Do this in order

The whole run stays inside the UI
1
Type your topic
Example: Strait of Hormuz
2
Choose the day window
Example: 7 days
3
Paste your n8n Webhook URL
Use the Test URL while building. Use the Production URL once your workflow is active.
4
Click “Test Connection”
This checks whether the URL is reachable and returns valid JSON.
5
Click “Run Workflow”
The UI sends the request and loads the returned markdown files automatically.
6
Edit and download
Review the files in the editor and export them when ready.
